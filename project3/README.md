        # 任务3：水电煤气管理系统

        ## 任务要求

        录入用户水电煤气数据、输出用户信息、修改用户信息、计算应缴费用。

        ## 样品特点

        - 界面风格：能源缴费仪表盘风
        - 支持新增、修改、删除、保存、重载。
        - 使用文件流保存数据到 `data/records.csv`。
        - 使用 `std::vector` 管理记录。
        - 使用 EasyX4MinGW 实现 C++ 图形界面。
        - 特殊功能按钮：`计算费用`。

        ## 字段

        - 用户姓名
- 用户编号
- 用水量
- 用电量
- 用气量
