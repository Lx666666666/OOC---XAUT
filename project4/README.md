        # 任务4：选课管理信息系统

        ## 任务要求

        输入、输出、修改、删除学生选课信息。

        ## 样品特点

        - 界面风格：教务卡片风
        - 支持新增、修改、删除、保存、重载。
        - 使用文件流保存数据到 `data/records.csv`。
        - 使用 `std::vector` 管理记录。
        - 使用 EasyX4MinGW 实现 C++ 图形界面。
        - 特殊功能按钮：`选课统计`。

        ## 字段

        - 学生姓名
- 学生学号
- 课程名称
- 任课教师
- 学分

        ## 编译

        ```powershell
        cd "C:\Users\lx\Desktop\project4"
        .\build.ps1
        ```

        生成程序：

        ```text
        Project4.exe
        ```

        ## 输入格式

        ```text
        请输入：学生姓名,学生学号,课程名称,任课教师,学分
例如：张三,20240101,C++程序设计,王老师,3
        ```
