        # 任务0：考勤管理系统

        ## 任务要求

        输入、输出、修改、删除学生缺课信息。

        ## 样品特点

        - 界面风格：校园考勤蓝白表格风
        - 支持新增、修改、删除、保存、重载。
        - 使用文件流保存数据到 `data/records.csv`。
        - 使用 `std::vector` 管理记录。
        - 使用 EasyX4MinGW 实现 C++ 图形界面。
        - 特殊功能按钮：`缺课统计`。

        ## 字段

        - 缺课日期
- 课程名称
- 学生姓名
- 学生学号
- 缺课类型
