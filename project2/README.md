        # 任务2：网吧管理系统

        ## 任务要求

        输入顾客、输出顾客信息、修改顾客信息、按会员类型和上机时间结账。

        ## 样品特点

        - 界面风格：网吧深色霓虹风
        - 支持新增、修改、删除、保存、重载。
        - 使用文件流保存数据到 `data/records.csv`。
        - 使用 `std::vector` 管理记录。
        - 使用 EasyX4MinGW 实现 C++ 图形界面。
        - 特殊功能按钮：`顾客结账`。

        ## 字段

        - 顾客姓名
- 顾客类型
- 机器号
- 上机小时
