        # 任务1：医院管理系统

        ## 任务要求

        输入病人、输出病人信息、修改病人信息、办理病人出院。

        ## 样品特点

        - 界面风格：医院绿白侧栏风
        - 支持新增、修改、删除、保存、重载。
        - 使用文件流保存数据到 `data/records.csv`。
        - 使用 `std::vector` 管理记录。
        - 使用 EasyX4MinGW 实现 C++ 图形界面。
        - 特殊功能按钮：`病人出院`。

        ## 字段

        - 病人姓名
- 病人类型
- 主治大夫
- 科室
- 入院日期
