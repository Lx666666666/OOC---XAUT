        # 任务8：活期储蓄账目管理系统

        ## 任务要求

        用户注册、登录、存款、取款和删除用户。

        ## 样品特点

        - 界面风格：银行沉稳金绿风
        - 支持新增、修改、删除、保存、重载。
        - 使用文件流保存数据到 `data/records.csv`。
        - 使用 `std::vector` 管理记录。
        - 使用 EasyX4MinGW 实现 C++ 图形界面。
        - 特殊功能按钮：`存取款`。

        ## 字段

        - 用户名
- 账号
- 密码
- 余额
